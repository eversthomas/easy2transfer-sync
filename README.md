# easy2transfer-sync
Wordpress Plugin um Daten aus easyVerein zu synchronisieren

## features
* holt sich alle Daten aus easyVerein, die eine bestimmte consent-ID haben
* im Backend kann ausgewählt werden, welche Felder in welcher Reihenfolge gerendert weden sollen
* öffentlich Kalender können gerendert werden
* es kann eine interaktive Karte der Mitglieder ausgegeben werden
